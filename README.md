# Presupuestos

Interfaz web + generación de PDF (Netlify Functions).

## Patentes / cliente en la nube (opcional)

Para sugerencias al escribir la patente y guardar datos tras generar un PDF, configura **Supabase** (plan gratuito suele bastar para pocos usuarios y poco almacenamiento). Pasos: **[docs/SUPABASE.md](docs/SUPABASE.md)**.

## Desarrollo

```bash
npm install
npm run dev
```
